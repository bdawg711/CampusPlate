CAMPUSPLATE SPRINT 5 — PREMIUM VISUAL OVERHAUL PRD
====================================================

VISION
------
Transform CampusPlate from a functional student nutrition app into a premium, luxury-feeling
product. The visual identity should evoke Virginia Tech prestige — think varsity letter jacket
meets modern fintech. Clean white space, Hokie maroon confidence, metallic silver precision,
metallic gold achievement. Every interaction should feel intentional, every animation earned.

Design philosophy: Professional sharpness over friendly softness. Architectural over bubbly.
Data-confident over data-decorated.

Metallic philosophy: Gold and silver are NOT flat colors. They are metallic gradients that
simulate how light catches brushed metal. BUT they are used with restraint — only on
achievement elements, ring accents, badges, grades, and milestone numbers. The majority of
the UI stays clean white + flat maroon + subtle borders. The metal appears on things you
EARN or things that represent DATA HIGHLIGHTS. That restraint is what makes it luxury, not gaudy.

================================================================================
METALLIC SHIMMER ANIMATION SPEC
================================================================================

The "shimmer" is a signature micro-animation used on achievement elements. It simulates
light catching a metallic surface.

How it works:
- A translucent white-to-transparent LinearGradient overlay (angled ~20deg)
- Gradient: [transparent, rgba(255,255,255,0.4), transparent]
- The gradient translates from left-to-right across the element over ~800ms
- Uses react-native-reanimated translateX animation
- Easing: Easing.inOut(Easing.quad) for smooth acceleration/deceleration

When to trigger shimmer:
- Badge unlock: plays ONCE when badge transitions from locked to unlocked
- Grade achievement: plays ONCE when daily score first loads as "A"
- Streak milestone: plays ONCE when streak number hits 7, 14, 30, 60, 100
- Goal hit: plays ONCE when a ring reaches 100% during logging animation
- NEVER loops continuously — shimmer is an event, not a decoration

Component: MetallicShimmer.tsx (see Phase 2 tasks)
- Accepts: width, height, borderRadius (to clip the shimmer to the parent shape)
- Renders as an absolute-positioned overlay
- Exposes a trigger() method via ref or a "play" boolean prop


================================================================================
METALLIC GRADIENT TEXT SPEC
================================================================================

For the "A" grade letter and milestone streak numbers, we need gradient-filled text.
React Native doesn't support CSS background-clip: text, so we use MaskedView.

Implementation pattern:
  import MaskedView from '@react-native-masked-view/masked-view';
  import { LinearGradient } from 'expo-linear-gradient';

  <MaskedView maskElement={<Text style={gradeStyle}>A</Text>}>
    <LinearGradient
      colors={['#8B6914', '#C5A55A', '#E8D5A3', '#C5A55A', '#8B6914']}
      locations={[0, 0.3, 0.5, 0.7, 1.0]}
      start={{x: 0, y: 0}}
      end={{x: 1, y: 1}}
    >
      {/* Invisible text to size the gradient */}
      <Text style={[gradeStyle, {opacity: 0}]}>A</Text>
    </LinearGradient>
  </MaskedView>

Where to use gradient text:
- Daily score "A" grade (only A — B through F stay flat colors)
- Streak number at milestones (7, 14, 30, 60, 100) — use gold gradient
- "Goal Complete!" celebration banner text — use gold gradient
- Nowhere else. Most text stays flat. Gradient text is a reward.

Dependency: @react-native-masked-view/masked-view
Install: npm install @react-native-masked-view/masked-view

CARD STYLE
----------
All cards should use:
- backgroundColor: 'white'
- borderWidth: 1, borderColor: 'border' (thin, subtle)
- borderRadius: 'm' (8px)

- Consistent padding: 'm' (16px) internal

Exception: Profile card on Settings and Daily Score card on Dashboard use borderRadius 'l' (10px).

TASK 3.1: Create SpiralRings component
File: src/components/SpiralRings.tsx

This is the signature visual element of the entire app. Four concentric SVG rings
representing calories, protein, carbs, and fat — each with a different rotational
start angle creating a spiral/portal effect.

Architecture:
- 4 SVG circles using strokeDasharray and strokeDashoffset
- Ring order (outer to inner): Calories (maroon), Protein (steel blue), Carbs (gold metallic), Fat (silver metallic)
- Each ring has a track (10% opacity of its base color) and a fill arc

METALLIC RING STROKES:
- Calories ring: flat maroon (#861F41) — brand identity, no gradient
- Protein ring: flat steel blue (#4A7FC5) — clean data color
- Carbs ring: SVG linearGradient using gold metallic stops
    <defs>
      <linearGradient id="goldMetallic" x1="0%" y1="0%" x2="100%" y2="100%">
        <stop offset="0%" stopColor="#8B6914" />
        <stop offset="30%" stopColor="#C5A55A" />
        <stop offset="50%" stopColor="#E8D5A3" />
        <stop offset="70%" stopColor="#C5A55A" />
        <stop offset="100%" stopColor="#8B6914" />
      </linearGradient>
    </defs>
    stroke="url(#goldMetallic)"
- Fat ring: SVG linearGradient using silver metallic stops
    Same pattern with silver gradient stops

Rotational offsets (start angles):
- Calories: 0deg (12 o'clock)
- Protein: -30deg (offset counterclockwise)
- Carbs: -60deg
- Fat: -90deg
- Applied via transform: rotate(Xdeg) on each SVG circle group

Fill-on-load animation:
- Uses useSharedValue for each ring progress (0 -> actual percentage)
- Time stagger: Calories starts immediately, Protein +120ms, Carbs +240ms, Fat +360ms
- Duration: outer 900ms, each inner 50ms shorter (900, 850, 800, 750)
- Easing: Easing.out(Easing.cubic)
- Cap visual fill at 95% of circle (leaves gap to show spiral even at 100%)

Over-target state:
- If a macro exceeds 100%, the ring fills to 95% AND shifts to gold metallic gradient
  (regardless of which ring it is — going over target = gold achievement)
- Add MetallicShimmer overlay on the over-target ring (plays once)

Update animation (when user logs a meal):
- Rings animate from current value to new value
- Brief "rewind" effect: ring moves back 5deg then surges forward
- Uses withSequence(withTiming(currentVal - 5, {duration: 150}), withSpring(newVal, {damping: 12, stiffness: 100}))
- Trigger haptic 'success' on any ring passing a 25% threshold

Center content:
- Calorie count as AnimatedNumber with ringCenter text variant
- "of X,XXX cal" subtitle in muted variant
- Both centered within the ring container

Props:
- calories: { current: number, goal: number }
- protein: { current: number, goal: number }
- carbs: { current: number, goal: number }
- fat: { current: number, goal: number }
- size?: number (default 280)
- animated?: boolean (default true)
